نظام إدارة الرياضيين - تطبيق ويب كامل بدون خوادم

طريقة الاستخدام:
1. افتح ملف index.html في أي متصفح.
2. استخدم أحد الأرقام التالية كرقم هوية:
   - رياضي: ATH1, ATH2, ..., ATH200
   - مدرب: COA1, COA2, ..., COA20
   - طبيب: DOC1, DOC2, DOC3, DOC4
   - مراقب مالي: FIN1
   - نائب مدير: DEP1
   - مدير: DIR1
3. لا يوجد كلمة مرور، فقط اضغط "دخول".
